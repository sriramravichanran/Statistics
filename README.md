# 📊 Statistics Case Study – Descriptive Analysis

This repository contains a **Statistics Case Study project** focused on **descriptive statistics** to understand data behavior and distribution.

The project demonstrates how basic statistical measures can be used to summarize and analyze a dataset effectively.

---  
   
## 📌 Project Overview 

In this case study, descriptive statistical techniques are applied to analyze the dataset and extract meaningful insights.
 
The analysis includes measures of central tendency, dispersion, and data range.
     
---   
## 📈 Statistical Measures Covered  
 
* Mean
* Median
* Mode  
* Standard Deviation
* Minimum Value
* Maximum Value
* Data Summary
